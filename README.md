# PRODIGY_DS_05

## Task 5 - Traffic Accident Data Analysis

This project analyzes traffic accident patterns based on weather conditions, road conditions, and time.

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Accidents by weather condition
- Accidents by road condition
- Accidents by hour
